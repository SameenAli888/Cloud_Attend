CloudAttend is a web-based system designed to automate online class attendance, validate lecture watch-time, monitor student engagement, and provide dashboards for both teachers and students.
This project is built using HTML, CSS, JavaScript, Node.js, and MySQL Workbench.

Project Overview

CloudAttend solves the problem of manual attendance and low engagement in online classes by providing automated attendance, focus tracking, and real-time monitoring.
